# Tech Stack

- Bun
- TypeScript
- Hono
- React
- Tailwind CSS v4
- oxlint
- oxfmt

# Commands

- `bun run dev`: Start the development server with watch mode.
- `bun run build`: Bundle the React SPA into `dist/client`.
- `bun run lint`: Run oxlint and fail on warnings.
- `bun run format`: Format files with oxfmt.
