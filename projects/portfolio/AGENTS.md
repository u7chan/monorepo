# AGENTS.md

## Tech Stack

- **Language**: TypeScript
- **Runtime**: Node
- **Package Manager**: Bun
- **Linter & Formatter**: Oxlint / Oxfmt
- **CSS Styling**: Tailwind CSS
- **Build & Bundler**: Vite
- **Backend**: Hono
- **Frontend**: React (SPA)
- **Testing**: Vitest
- **Routing**: TanStack Router
- **Database**: PostgreSQL
- **ORM**: Drizzle ORM
- **Dev Environment**: Dev Containers (VSCode)

## Code style

- TypeScript strict mode
- Single quotes, no semicolons
- Tab width: 2 spaces
- Print width: 120
- Trailing comma: es5
- End of line: lf

## Linting & Formatting

- **Lint**: `bun run lint`
- **Format**: `bun run format`

## Test

- **Test**: `bun run test`
- **Test Coverage**: `bun run test:coverage`
