# hono-openai-client

## Quick Start

OpenAI:

```sh
API_KEY="<YOUR_OPENAI_API_KEY>" \
BASE_URL="https://api.openai.com/v1" \
MODEL="gpt-4o-mini" \
bun run dev
```

DeepSeek:

```sh
API_KEY="<YOUR_DEEPSEEK_API_KEY>" \
BASE_URL="https://api.deepseek.com" \
MODEL="deepseek-chat" \
bun run dev
```

[http://localhost:3000](http://localhost:3000)
