# python_docling

see: https://github.com/DS4SD/docling

## usage

1. Change directory

    ```sh
    cd projects/python_docling
    ```

1. Install packages

    ```sh
    uv sync --frozen
    ```

1. Copy path

    ```sh
    projects/python_docling/test-data/c4611_sample_explain.pdf
    ```

1. Show [Run and Debug] view
1. Select `python_docling`
1. Start Debugging (F5)
