# python_word_analyzer

## usage

1. Change directory

    ```sh
    cd projects/python_word_analyzer
    ```

1. Install packages

    ```sh
    uv sync --frozen
    ```

1. Copy path

    ```sh
    projects/python_word_analyzer/test-data/sample.docx
    ```

1. Show [Run and Debug] view
1. Select `python_word_analyzer`
1. Start Debugging (F5)
